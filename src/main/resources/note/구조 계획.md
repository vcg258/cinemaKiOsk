src/main/resources/
│
│  ※ 표기 방식
│    ├─ 파일명  ← [Controller]  METHOD /url
│    ├─ 메서드: 접근자 반환타입 메서드명(파라미터)
│    └─ Model : 뷰에 전달해야 할 데이터 (백엔드 작업자 참고)
│
│  ※ GET+POST 둘 다 있는 페이지: 화면 조회(GET) + 처리 후 Redirect(POST)
│  ※ DTO 흐름: 각 화면은 이전 단계의 DTO를 받아서 다음 단계로 넘긴다
│              영화목록 → MovieDTO → 스케줄 → ScheduleDTO → 좌석 → ReservationDTO → 결제
