총 관리자 - 모든걸 볼 수 있고 관리 할 수 있어야하는걸 기준으로 두겠음


1. 기본 화면
관리자 페이지는 이벤트 없어도 화면보호기로 안가게 설정 필요.

2. 영화 목록
영화 삭제시 웹창에서 바로 제거 되는게 아닌 상태가 삭제예정으로 변경 상영 종료일도 문구 추가.

상영 시작 시간과 종료시간이 보여주면 좋을거같음 관리자 화면만 봤을때는 이 영화의 개봉일 종영일을 알 수 없음.

영화가 상영종료가 되면 뷰에서 사라지는게 맞음 하지만 기록(로그)로는 볼 수 있어야함 전체 로그

3. 영화 상영 관리
현재는 스케줄 등록하기가 굉장히 힘든 구조로 되어있음(서버에서 시간이 겹치면 막히게 구현 됨)
상영관별 이미 등록된 스케줄이 있다면 표시 or 선택X(해당 일자 스케줄이 꽉 찼을 경우) 로 하면 좋아보임

시작시간만 받아서 계산해서 종료시간을 넣는데 현재 등록된 영화를 선택해서 지정한 스케줄에 
무슨 영화를 하는 스케줄인지 종료시간은 영화의 런타임 + 정리시간(계산 값은 이미 구현 되어 있기때문에 DB에서 값만 가져오면 됨)을 계산해서 보이게 하면 좋아보임
또한 상영관의 좌석도 보여지게 해놓으면 좋겠긴함

상영관별 마지막 스케줄 종료시간을 기준으로 시작시간에 값을 자동으로 넣어주면 편해보임

스케줄 상태도 보여지게 만료여부 삭제는 만료처리 -> 삭제

그리고 기간을 설정해서 조회도 할 수 있게 하면 좋아보임

4. 상영관 목록
상영관 정리시간 표시와 수정하는 곳이 있지않음 추가 필요함

5. 좌석 목록, 좌석 정책 수정 (미완)


6. 정책 목록
할인정책 적립정책 이렇게 두가지 유형이 있기때문에 이건 상의가 필요해보임

7. 환불 처리
환불 처리가 됐을때 환불가능 -> 환불완료

이미 환불이 완료된 내역을 조회했을때 아래쪽 문구에 "이미 환불처리된 내역입니다" 라고 띄워두면 좋아보임

8. 일일 통계
범위를 늘리면 통계수가 너무 많음 페이징처리 (최고 후순위 사실 없어도 됨)

9. 회원 정보 관리
전체 로그 리스트 볼수있는 기능과 지정 회원을 누르면 포인트 전체 내역을 볼수있게, 그리고 이미 밖에서 회원 정보를 다 볼 수 있음 상세정보 버튼의 필요성이 존재한가?

5. 좌석 정책 수정 -> 비활성화
